Overview

This is a full-stack Bookstore web application built using the MERN stack (MongoDB, Express.js, React, Node.js). The application allows users to browse books, view details, register/login, and manage purchases. It also includes an admin panel to manage books and users.

Features
*User Features
-User registration and login (JWT authentication)
-Browse all books
-Search books by title or category
-View book details
-Add books to cart
-Place orders

*Admin Features
-Add new books
-Update existing books
-Delete books
-View all users
-Manage orders

Tech Stack
